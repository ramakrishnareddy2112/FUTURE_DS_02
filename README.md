# Customer Churn Analysis

## Project Overview
This project focuses on analyzing customer churn using the Telco Customer Churn dataset. The goal of this analysis is to identify patterns and factors that influence customers to leave a telecom service.

Understanding churn behavior helps businesses take preventive measures to improve customer retention and overall service quality.

---

## Dataset
The dataset used in this project is the **Telco Customer Churn dataset**.

It contains customer information such as:

- Customer ID
- Gender
- Contract Type
- Internet Service
- Payment Method
- Tenure
- Monthly Charges
- Total Charges
- Tech Support
- Customer Churn (Yes / No)

## Data Analysis

The dataset was analyzed using Python to understand churn patterns. Several visualizations were created to identify trends and relationships between customer features and churn behavior.

The analysis includes:

- Customer churn distribution
- Churn by contract type
- Churn by payment method
- Churn by internet service
- Churn by tech support
- Customer tenure distribution
- Monthly charges vs churn

---

## Key Insights

1. Customers with **month-to-month contracts** have the highest churn rate.
2. Customers using **electronic check payment methods** tend to churn more frequently.
3. Customers **without technical support services** show higher churn behavior.
4. Customers with **shorter tenure periods** are more likely to leave the service.
5. Customers with **higher monthly charges** tend to churn more.
