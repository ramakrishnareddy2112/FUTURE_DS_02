# Customer Churn Analysis (FUTURE_DS_02)

## Project Overview
This project analyzes customer churn using the Telco Customer Churn dataset. The goal is to identify patterns and factors that influence customer churn and provide insights that can help improve customer retention.

Customer churn analysis helps businesses understand why customers leave their services and how to reduce churn by improving customer satisfaction and retention strategies.

---

## Dataset
The dataset used in this project is the **Telco Customer Churn dataset**.

It includes information such as:
- Customer ID
- Gender
- Tenure
- Contract Type
- Payment Method
- Monthly Charges
- Total Charges
- Churn Status

Dataset file:

---

## Key Insights

- Customers with month-to-month contracts have a higher churn rate.
- Customers using electronic check payment methods churn more frequently.
- New customers with low tenure are more likely to churn.
- Higher monthly charges are associated with higher churn.

---

## Recommendations

- Encourage customers to switch to long-term contracts.
- Improve customer support services.
- Offer loyalty programs for new customers.
- Provide discounts for customers with high monthly charges.

---

## Conclusion

This project demonstrates how data analysis and visualization can help identify important factors affecting customer churn. The insights obtained can help businesses improve customer retention strategies and enhance overall customer satisfaction.
